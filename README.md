## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

Run seed data script:
```sh
npm run seed
```

Run Webpack
```sh
npm run dev-react
```

Start Node Server:
```sh
npm start
```
